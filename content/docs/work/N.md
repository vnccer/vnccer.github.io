面对公司试图以“不胜任”为由的不合理辞退，切勿盲目签字或情绪化应对。请严格按照以下指南保护自身权益：

## 一、 核心应对口诀
* **不主动提离职：** 只要自己提交辞职信，任何经济补偿都会归零。
* **不承认不胜任：** 拒绝签署任何不符合事实的考核表或绩效改进计划（PIP）。
* **不拒绝合理工作：** 保持正常打卡上下班，不给公司留下“旷工”违纪的把柄。
* **坚持法定赔偿：** 坚守 N+1 或 2N 的法律底线，不接受忽悠。

## 二、 必须收集的“保命”证据
* **劳动合同存证：** 拍摄合同的完整页面（含封面、条款页及双方签字盖章页），确保字迹清晰。
* **约谈全程录音：** 与 HR 或领导的所有谈话必须偷偷录音。重点引导对方承认“想招更有经验的人”或“直接让你走”。
* **日常履职证明：** 备份周报、月报、工作群沟通记录、客户好评或合格的绩效邮件，证明自己能够胜任工作。
* **公司招聘截图：** 截图并保存公司在各大平台招聘你当前岗位的启事，证明岗位依然存在，是公司想“换人”而非“裁岗”。

## 三、 警惕公司的三大陷阱
1. **诱导“和平分手”：** 领导以“被辞退不好看、影响下份工作背调”为由，劝你主动提离职。**坚决拒绝。**
2. **强签“莫须有”的考核：** 公司单方面拿出不达标的考核让你签字。**若被强制要求签字，必须在签名旁明确备注：“指标未经确认，本人对考核过程和结果不予认可。”**
3. **口头通知滚蛋：** HR 口头让你“明天不用来了”。**口头通知无效，要求公司出具加盖公章的正式《解除劳动合同通知书》，拿到书面证据前继续按时打卡。**

## 四、 明确你的赔偿底线
| 辞退性质     | 赔偿标准  | 判断依据                                                     |
| :----------- | :-------- | :----------------------------------------------------------- |
| **合法辞退** | **N + 1** | 公司严格完成了“考核不达标 ➔ 培训或调岗 ➔ 再次考核不达标”的法定流程，且未提前30天书面通知。 |
| **违法辞退** | **2N**    | 公司未按法定流程操作，或仅以“需要更有经验的人”为借口直接将你辞退。 |

## 五、 合理辞退流程参考（公司需履行的法定程序）
如果公司要依据《劳动合同法》第40条第2项以“不胜任工作”为由合法辞退员工，必须形成完整的证据链，严格走完以下所有流程（缺一不可，否则即为违法解除）：
1. **第一次认定不胜任：** 具备合理、明确的考核标准，且有员工签字确认的“不达标”考核结果。
2. **法定补救措施（二选一）：** 必须对员工进行针对性的专门**培训**，或者对其进行**调岗**（调岗需具备合理性，不能具有侮辱性或恶意降薪）。
3. **第二次认定不胜任：** 培训结束后或在新的工作岗位上，经过再次考核，员工依然无法达到合理的工作要求。
4. **提前通知与经济补偿：** 必须提前30天以书面形式通知劳动者本人（如需立即解除，则额外支付1个月工资作为代通知金，即N+1），并依法支付经济补偿金（N）。
5. **通知工会（程序性要求）：** 单方面解除劳动合同前，应当事先将解除理由通知公司工会。
