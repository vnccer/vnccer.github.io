# 一、AI 工具全生态分类表 (2026版)

| 工具大类               | 细分类别          | 代表产品                           | 核心逻辑                       | 定价模式                |
| :--------------------- | :---------------- | :--------------------------------- | :----------------------------- | :---------------------- |
| **应用层 (SaaS)**      | **官方对话应用**  | ChatGPT, Claude, 豆包, Kimi        | 官方“旗舰店”，功能最稳、最新。 | 订阅制 ($20/月)         |
|                        | **聚合客户端**    | Poe, Monica, ChatBox, NextChat     | 一个界面调动所有模型。         | 订阅制 或 免费(自带Key) |
| **生产力层 (Tooling)** | **AI IDE / 插件** | Cursor, Trae, Copilot, Windsurf    | AI 直接读取代码上下文。        | 订阅制                  |
|                        | **CLI 命令行**    | Claude Code, Aider, Codex CLI      | 在终端直接改代码、下指令。     | API 消耗 或 订阅        |
| **资源层 (Infra)**     | **原厂 API 平台** | OpenAI Platform, DeepSeek 开放平台 | 直接对口开发者。               | 按量计费 (Token 计费)   |
|                        | **API 中转/分发** | 柏拉图 API, OpenRouter, OhMyGPT    | 解决跨境支付和多模型统一。     | 预充值/按量计费         |
| **自建层 (Self-host)** | **本地运行环境**  | Ollama, LM Studio                  | 在自己电脑上跑模型，断网可用。 | 免费 (消耗自己的显卡)   |
