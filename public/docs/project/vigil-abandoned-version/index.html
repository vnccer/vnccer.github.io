      ж ёеҝғе‘ҪйўҳпјҡеҸҜи§ЈйҮҠзҡ„еҠ еҜҶжөҒйҮҸеҲҶзұ»гҖӮж”ҜжҢҒдёӨжқЎжЈҖжөӢи·ҜзәҝеңЁзӣёеҗҢж•°жҚ®дёҠзҡ„е…¬е№іеҜ№жҜ”пјҡ

      и·Ҝзәҝ иҫ“е…Ҙ жЁЎеһӢ еҸҜи§ЈйҮҠжҖ§
      з»Ҹе…ё ML 30 з»ҙзү№еҫҒпјҲJA4+SPL+IAT жҲ–еӯ—иҠӮз»ҹи®Ўпјү XGBoost SHAP TreeExplainer

      4.2 йў„еӨ„зҗҶи„ҡжң¬ (preprocess.py)#

      ж ёеҝғ CLIпјҡ

      # XGBoost

# еҸӮж•°: --max-files N --output-dir ./out --label-file labels.json

      CSTNET ж•°жҚ®йӣҶиҮӘеҠЁжҢү PCAP вҶ’ TSV вҶ’ NPY дјҳе…Ҳзә§еӣһйҖҖжү«жҸҸгҖӮTSV ж јејҸдёә label<TAB>hex_textпјҲhex зј–з Ғеӯ—иҠӮжөҒпјүпјҢNPY ж јејҸдёәйў„ token еҢ–зҡ„ fine-tuning еҲҶзүҮгҖӮ

      4.3 ж•°жҚ®йӣҶ Г— жЁЎеһӢе…је®№зҹ©йҳө#

      ж•°жҚ®йӣҶ XGBoost ET-BERT еӨҮжіЁ
      CIC-IDS-2017 (CSV) 77 з»ҙеҺҹе§Ӣзү№еҫҒ дёҚж”ҜжҢҒ ж— еҺҹе§ӢиҪҪиҚ·еӯ—иҠӮ
      CSTNET-TLS1.3 (TSV) 30 з»ҙеӯ—иҠӮз»ҹи®Ўзү№еҫҒ 512 з»ҙ byte token flow + packet еҸҢзә§еҲ«
      Sandbox (.pcap) 30 з»ҙ JA4+SPL+IAT 512 з»ҙ byte token VigilFeatureExtractor жҸҗеҸ–
      еҗҲжҲҗж•°жҚ® 30 з»ҙйҡҸжңә 512 з»ҙйҡҸжңә token д»…йӘҢиҜҒжөҒж°ҙзәҝ

      4.4 ET-BERT Token и§„иҢғ#

      • иҜҚиЎЁ: [PAD]=0, byteвҶ’id=byte+1 (0x00вҶ’1, 0xFFвҶ’256), [CLS]=257, [SEP]=258, vocab=259
      • еәҸеҲ—: [CLS] + pkt1[0:60] + [SEP] + pkt2[0:60] + [SEP] + ... + [PAD]*N
      • жңҖеӨ§й•ҝеәҰ: 512пјҲжҒ°еҘҪеЎ«ж»Ў 8 еҢ… Г— 61 еӯ—иҠӮ + 1 CLS + 23 PADпјү

      дә”гҖҒжҠҖжңҜж Ҳ#

      еұӮзә§ жҠҖжңҜ з”ЁйҖ”
      API FastAPI + Uvicorn REST зҪ‘е…і
      йҷҗжөҒ Redis / еҶ…еӯҳ ж»‘еҠЁзӘ—еҸЈ
      жөҒйҮҸи§Јжһҗ Scapy .pcap вҶ’ жөҒиҒҡеҗҲ
      з»Ҹе…ё ML XGBoost + scikit-learn еҹәзәҝеҲҶзұ»еҷЁ
      ж·ұеәҰеӯҰд№  PyTorch 2.6 + Transformers ET-BERT (fp16, CUDA 12.4)
      еҸҜи§ЈйҮҠжҖ§ SHAP / InputГ—Gradient еҸҢеҸҷдәӢиҫ“еҮә
      eBPF BCC TCP жөҒзӣ‘жҺ§ (Linux)
      жІҷз®ұ Docker SDK йҡ”зҰ»жү§иЎҢ
      иҜ„дј° NumPy + asyncio + aiohttp жҢҮж Ү + еҺӢжөӢ
