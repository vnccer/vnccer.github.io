    用户名：admin 密码：melehifokivai roles：admin-gui和manager-gui

    三、提权#

    sudo -l要求提供密码,因此检查其他目录

    `find / -perm -u=s -type f 2>/dev/null`

    出来巨多文件（目前不适合用） 在/home/文件夹中有两个用户jaye和randy，其中jaye需要权限

    ssh jaye@192.168.203.140

    新开ssh终端连接到jaye，再输入bash切换为更优美的界面 找到look文件，使用方法见点击跳转look使用说明
